JSON - это по сути текст, который описывает данные

1. Значение в JSON это имя:значение
2. Массив - это данные обёрнутые в квадртыне скобки [ и ]
3. Объект - это данные обёрнутые в фигурные скобки { и }

имя : значение

"имя" ВСЕГДА указывается в кавычках ""
Значение может быть:
- Числовые (целые или дробные с точкой)
- Строковые (обвернутые в двойные кавычки)
- Логические (true или false)
- Нулевое значение (null)
- Другие массивы (обвернутые в квадратные скобки [ и ])
- Другие объекты (обвернутые в фигурные скобки { и })

Значения одного уровня разделяются запятыми

Для работы с JSON в PHP есть команды:

json_decode - преобразовывает строку в объект/массив
json_encode - преобразовывает объект/массив в строку
json_last_error - возвращает результат преобразования
